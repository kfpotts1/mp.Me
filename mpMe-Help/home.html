mp.Me - Deleting Files

Note: mp.Me will NEVER attempt to delete your files.

The delete screen won't actually delete files. It will simply search a directory and any sub-directories found and move all files that have not been touched since a set date to a folder called "suggestedDelete". Then, you can browse the created folder or delete it in one action!



On this screen you will see two rows and a button.

The first row reflects the directory selected in the options screen.

The second row reflects the date set in the options screen. This is known as the "date tolerance". The date tolerance essentially says "Find files that have not been touched since this date to be deleted".




WARNING: mp.Me assumes a level of user responsibility. The software will not check if the directory selected is the root directory, and can critically impair a computer's ability to function if used incorrectly. We trust the user to use this software in it's intended manner.

mp.Me - Renaming Files

On this screen you will see three rows and a button.

The first row reflects the directory selected in the options screen.

The second row reflects the file naming convention input to search for.

The third row reflects the file naming convention output to search for.



More information on these settings and what they do can be found further down in the page in the Options section.




mp.Me - Setting Options

This screen contains all the changable settings allows in mp.Me. This is the only menu where settings can be changed.

The first row reflects the directory selected in the options screen.

The second row allows you to change the file naming convention input to search for. For example, if we were looking for files that looked like 01__Adele__Hello__25 , we would give a file naming convention input of trackNumber__artistName__trackName__albumName .

The third row reflects the file naming convention output to search for. For example, if we wanted to name the file above to 01__Hello , then we would give a file naming convention output that looks trackNumber__trackName .

The next row contains the settings for date tolerance and operating system. The date tolerance essentially says "Find files that have not been touched since this date to be deleted"; this needs to be in a MM/DD/YYYY format. The operating system lets our software know what file commands to use in order to run on your system.


If you are ever confused or cannot get something to work, a good first step would be resetting the options file. This can be done by pressing the "Reset to Default" button in the bottom right.
